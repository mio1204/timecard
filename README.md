## 使い方

---

1. ユーザー登録後、打刻ページに移動します。
2. 勤務状況に応じてボタンを押下するとその時刻が記録されます。
3. 休憩は開始と終了を同日中に繰り返し記録できます。
4. 「日付一覧」「ユーザー一覧」ページから個人の勤怠を確認できます。
