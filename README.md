# **Timecard**

社員間で共有可能な勤怠アプリです。

※こちらは Coachtech の提出課題で作成したものです。

## 使い方

---

1. ユーザー登録後、タイムカードページに移動します。
2. 勤務状況に応じてボタンを押下します。
3. 休憩は開始と終了を同日中に繰り返し行えます。
4. 「日付一覧」「社員一覧」ページから勤怠状況を確認します。
